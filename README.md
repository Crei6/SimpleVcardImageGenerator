# VcardQRCodeGenerator
## What is it?
Mit dem simplen Tool, kann man einen QR-Code generieren lassen, welcher beim Scan deine von dir angegeben Daten als Kontakt auf dem scannenden Gerät zum sofort abspeichern anzeigt. Die Daten die diese sogenannte "vCard" (-Visitenkarte) in dem QR-Code speichert beinhaltet folgende:
Vorname, 
Nachname, 
Telefonnummer, 
Email.
Hochgeladen damit man kostenlos ganz einfach ohne Abo sich diese vCard erstellen kann.


## How to install
1. Lade das Projekt als .zip File herunter.
2. Öffne das Projekt mit Visual Studio deiner Wahl.
3. Öffne Program.cs und ändere den string "filePath" zu einem Pfad deiner Wahl. Das fertige Bild wird dann in diesem Pfad mit dem von dir gewähltem Namen abgespeichert.
4. Starte das Projekt mit [F5] oder per Start von Visual Studio.
5. Ein Konsolenfenster öffnet sich. Befolge die Hinweise darin.
